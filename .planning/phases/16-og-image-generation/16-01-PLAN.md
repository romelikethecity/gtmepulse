**Generator script (scripts/generate_og_images.py):**

Create a standalone script that can be imported by build.py OR run directly.

Core function: `generate_og_images(pages_data, output_dir, templates_dir)`
- `pages_data`: list of dicts with keys: `rel_path`, `title`, `subtitle`, `template` (template name without .html), `og_filename`
- `output_dir`: path to output/ directory
- `templates_dir`: path to og-templates/ directory

Implementation:
1. Load all template HTML files from `templates_dir` into a dict keyed by template name (strip .html)
2. Launch ONE Chromium instance via `playwright.sync_api.sync_playwright()`
3. Create ONE page with viewport `{"width": 1200, "height": 630}`
4. For each page_info in pages_data:
   a. Select template (fall back to "og-default" if specified template not found)
   b. Replace `{{TITLE}}`, `{{SUBTITLE}}`, `{{CATEGORY}}` in template HTML
   c. Call `page.set_content(html, wait_until="networkidle")`
   d. Call `page.screenshot(path=og_path)` where og_path is `output_dir/assets/og/{og_filename}`
5. Close browser
6. Print summary: "Generated {N} OG images in {elapsed:.1f}s"

Add helper functions:
- `og_filename_from_path(rel_path)` — converts `salary/junior/index.html` to `salary-junior.png`, `index.html` to `index.png`
- `og_template_for_path(rel_path)` — maps path prefixes to template names:
  - `salary/` -> `og-salary`
  - `tools/` -> `og-tool`
  - `insights/` or `blog/` -> `og-article`
  - `glossary/` -> `og-glossary`
  - everything else -> `og-default`

Add `--skip-og` awareness: the function should accept an optional `skip` parameter that prints "Skipping OG generation" and returns early.

Add `if __name__ == "__main__":` block for standalone testing that generates OG images for a small test set (3-5 pages) to verify templates render correctly.

       OG_PAGES.append({
           "rel_path": rel_path,
           "title": title,
           "subtitle": subtitle,
           "template": og_template_for_path(rel_path),
           "og_filename": og_filename_from_path(rel_path),

4. Rather than modifying every individual build_* function, add a BULK registration step in `main()` that scans ALL_PAGES after all pages are built. This is cleaner than touching 80+ build functions:
   ```python
   # In main(), after all build_* calls and before build_sitemap():
   print("\n  Registering OG pages...")
   for rel_path in ALL_PAGES:
       # Extract title from the generated HTML file
       filepath = os.path.join(OUTPUT_DIR, rel_path)
       with open(filepath, "r", encoding="utf-8") as f:
           html = f.read()
       title_match = re.search(r'<meta property="og:title" content="(.*?)"', html)
       title = title_match.group(1).replace(" - GTME Pulse", "") if title_match else "GTME Pulse"
       # Extract description for subtitle
       desc_match = re.search(r'<meta name="description" content="(.*?)"', html)
       subtitle = desc_match.group(1)[:80] if desc_match else ""
       register_og(rel_path, title, subtitle)
   ```

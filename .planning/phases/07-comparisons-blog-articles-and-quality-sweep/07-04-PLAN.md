Enhance the existing validate_pages() function (around line 8377) to perform ALL QUAL2 checks. The current validator checks: title length, description length, H1 count, em-dashes, banned words. Add these checks:

**QUAL2-01 (SEO metadata):**
- Canonical URL exists: check for `<link rel="canonical"` tag
- OG tags exist: check for `og:title`, `og:description`, `og:url`, `og:image`
- Twitter Card tags exist: check for `twitter:card`, `twitter:title`, `twitter:description`

**QUAL2-02 (BreadcrumbList schema):**
- Check for `BreadcrumbList` in JSON-LD script tags on every page except homepage, privacy, terms, 404
- Warn if missing

**QUAL2-03 (Internal links):**
- Count internal links (href starting with "/" but not in nav/footer) in the main content area
- Detect the main content by looking for links between the breadcrumb and the footer
- Warn if fewer than 3 internal links in content body (skip homepage, privacy, terms, 404, newsletter)

**QUAL2-06 (FAQPage schema):**
- For comparison pages (path contains "comparisons/" and not index): check for `FAQPage` in JSON-LD
- Warn if comparison page missing FAQPage schema

**QUAL2-07 (Duplicate detection):**
- Collect all titles and descriptions across all pages
- After processing all files, check for duplicates
- Warn on any duplicate title or duplicate description

**QUAL2-08 (Source citations):**
- For pages under salary/, careers/, tools/, benchmarks/, comparisons/, blog/ (not indexes): check for "State of GTM" or "source-citation" class
- Warn if data page missing source citation

**QUAL2-09 (Writing standards - already partially done):**
- The existing em-dash and banned word checks cover this
- Add check for false reframe patterns: search for regex patterns like "not X, it's Y" / "isn't X\. It's Y" / "less about .* more about"
- Warn on any matches

**QUAL2-04 and QUAL2-05 (Word counts):**
- For pages under salary/, careers/, tools/, benchmarks/: estimate word count from text content (strip HTML tags, count words). Warn if under 1,200.
- For pages under blog/: estimate word count. Warn if under 1,500.
- Skip index pages (just the index.html directly in each section root).

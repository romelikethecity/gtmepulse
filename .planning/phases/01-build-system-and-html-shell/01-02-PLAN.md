**Content standards validator** - `validate_pages()` function called after all pages built:
- Scan all HTML files in output/ directory
- For each file, extract:
  - `<title>` tag content -> check length is 50-60 chars (the full title including " - GTME Pulse")
  - `<meta name="description" content="...">` -> check length is 150-158 chars
  - Count `<h1>` tags -> warn if != 1
  - Check for em-dash character (U+2014, \u2014) -> warn if found
  - Check for banned words from CLAUDE.md list (robust, leverage, synergy, holistic, cutting-edge, seamless, game-changer, paradigm shift, revolutionary, genuinely, truly, really, actually, quite, extremely, unlock, unleash, enhance, exceed, empower, supercharge, harness, spearhead, delve, landscape, tapestry, frontier, resonates, positioning) -> warn if found
- Print warnings but do NOT fail the build (WIP pages may have temporary content)
- Print "Content validation: X warnings" or "Content validation: all clear"

Note on CONTENT-06 (no false reframes): This is a writing convention that cannot be automated. It is enforced during content authoring per CLAUDE.md rules.

**Sitemap generator** - `build_sitemap()`:
- Iterate ALL_PAGES list
- Convert index.html paths to clean URLs (salary/junior/index.html -> /salary/junior/)
- Generate XML with `<url><loc>{SITE_URL}{clean_path}</loc><lastmod>{BUILD_DATE}</lastmod></url>` for each
- Write to output/sitemap.xml (use direct file write, NOT write_page -- sitemap should not be in ALL_PAGES)

**Robots generator** - `build_robots()`:
- 4 lines: `User-agent: *\nAllow: /\n\nSitemap: {SITE_URL}/sitemap.xml`
- Write to output/robots.txt

**main() pipeline** (exact order):

7. Write CNAME: single line "gtmepulse.com" to output/CNAME
8. Run validate_pages()
9. Print summary: total pages from len(ALL_PAGES), output dir path, preview command
